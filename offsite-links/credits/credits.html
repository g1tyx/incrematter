Incrematter credits


This game was inspired by various incremental game developers, such as Hevipelle, dan-simon, etc. and games like Antimatter Dimensions and FE000000.

This game uses ExpantaNum.js (there's no need for this honestly) created by Naruyoko. This library allows for numbers much, much bigger than the standard JavaScript limit (>>> 1.78e308).


Everything else in this game is made by me.
